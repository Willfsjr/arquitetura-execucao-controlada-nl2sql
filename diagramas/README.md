# Diagramas

Este diretório reúne os diagramas da arquitetura proposta no TCC.

## Diagramas sugeridos

- diagrama geral do pipeline NL2SQL
- diagrama de componentes
- fluxo de decisão allow/deny
- fluxo de auditoria e evidência
- comparação entre baseline inseguro e arquitetura controlada

## Pipeline resumido

Usuário → LLM → intenção estruturada → normalizador/resolvedor → policy engine → compilador por templates → executor controlado → resposta com evidência

## Objetivo

Apoiar a documentação técnica, a escrita do TCC e a apresentação final.
