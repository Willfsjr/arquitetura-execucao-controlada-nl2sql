# Protótipo

Este diretório será utilizado para armazenar a implementação experimental da arquitetura proposta.

## Objetivo

Construir um protótipo controlado que permita comparar:
- baseline inseguro
- proteções superficiais
- políticas estruturais
- arquitetura alvo com execução mediada

## Componentes esperados

- parser da intenção estruturada
- normalizador de datas e métricas
- policy engine
- compilador de templates SQL
- executor controlado
- módulo de auditoria

## Relação com o TCC

O protótipo será a base da avaliação experimental, permitindo medir risco, utilidade, latência e governança em diferentes configurações.
